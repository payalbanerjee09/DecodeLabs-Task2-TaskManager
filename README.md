# Enterprise Task Architecture Dashboard ✨

Welcome to the **Enterprise Task Architecture Dashboard**, engineered as part of **Task 2 / Week 2** for the Full Stack Development Internship at **Decode Labs**. 

This application bridges the gap between structured corporate task allocation and a beautiful, intuitive, and modern pastel-pink aesthetic user interface.

## 🚀 Key Features

- **State Persistence (LocalStorage Integration):** Automatically commits data states to the browser memory, ensuring no structural data loss upon hard page reloads.
- **Dynamic Runtime Query Filtering:** Implements advanced JavaScript array manipulation (`.filter()`) to instantly sort tasks between *All*, *Pending*, and *Completed* targets without server requests.
- **Real-Time Data Aggregation:** Active system metrics integrated at the footer level to dynamically calculate progress percentage and task counting.
- **Event-Driven Task Injection:** Dual-mode execution enabling task creation through explicit mouse clicks or automated keyboard event triggers (`Enter` key handler).
- **Visual Hierarchy badging:** Features priority categorization (High, Medium, Low) wrapped inside conditional color-coded UI badges.

## 🛠️ Tech Stack & Architecture

- **HTML5:** Semantic architecture layout.
- **CSS3:** Custom Glassmorphism styles, fluid gradients, layout flexibility, and responsive configurations.
- **Vanilla JavaScript (ES6):** State array management, filter query engine, event delegation, and browser API interaction.
